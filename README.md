### Project tile: Predict Home Prices with Python and Linear Regression.

#### Goal: Using Pythons library to predict the price of housing in the near futures bases on pattern.

##### Component:

###### 1. Linear Regression:

Linear regression in statistics is a method used to understand the relationship between independent variables (like house size) and a dependent variable (such as house price)
by fitting a straight-line equation to the data. In simple terms, it involves drawing a line through data points to describe patterns and make predictions.

In data science, it’s commonly applied to tasks like forecasting sales, examining economic patterns, and evaluating how different factors influence outcomes.

In this project, to build a linear regression model that estimates house prices based on their sizes. By using a scatterplot of sample data, the model will identify the 
relationship between these variables and use it to generate predictions.

###### 2. Libraries use:

🔢 NumPy provides a powerful base for performing numerical computations and working with data efficiently.

🐼 Pandas enables you to inspect, clean, explore, and manipulate datasets from a variety of sources.

📈 Matplotlib helps convert data into clear visual representations such as 2D plots and bar charts.

The dataset being uses for this project compares house size and house prices of properties recently sold in Brooklyn's Dumbo neighborhood to predict the cost of houses based on size.
